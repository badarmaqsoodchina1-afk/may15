Personal academic homepage for GitHub Pages.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main webpage |
| `style.css` | Styles |
| `profile.jpg` | **Add your photo here** (rename your photo to this) |

## Setup: Add your profile photo

1. Copy your photo into this folder and name it **`profile.jpg`**
2. That's it — the page will display it automatically.

## Deploy to GitHub Pages (step-by-step)

1. Create a GitHub account at [github.com](https://github.com) if you don't have one.
2. Create a new repository named exactly: **`<your-username>.github.io`**
   - Example: if your GitHub username is `sanaullahbadar`, the repo must be `sanaullahbadar.github.io`
3. Upload all files in this folder to that repository.
4. Go to **Settings → Pages** in your repo and set source to `main` branch / `/ (root)`.
5. Your site will be live at `https://<your-username>.github.io` within a few minutes.

## Updating your links

Open `index.html` and update these placeholders:
- **Google Scholar**: replace `XXXXXXX` in the Scholar URL with your actual user ID
- **LinkedIn**: replace `/in/` with your full LinkedIn profile URL
- **ResearchGate**: replace `/profile/` with your full ResearchGate profile URL
